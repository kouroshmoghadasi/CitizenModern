Ш®ШұЩҲШ¬

ШӘШәЫҢЫҢШұ ШұЩ…ШІ Ш§ШҜЩ…ЫҢЩҶ

Ш§ЩҒШІЩҲШҜЩҶ Ъ©Ш§ШұШЁШұ

Щ…ШЁЩ„Шә Ш§ШҙШӘШұШ§Ъ© (ШҜЩ„Ш§Шұ)

Ш§ЫҢЩҶ Щ…ШЁЩ„Шә ШҜШұ ШіШ§ЫҢШӘ ЩҲ ШҜШұ ЩҒШұЩ… ШІЫҢШұ ШЁЩҮвҖҢШөЩҲШұШӘ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ШҜШұ Ш«ШЁШӘ ЩҫШұШҜШ§Ш®ШӘ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…ШЁЩ„Шә ЩҮШұ ЩҫШұШҜШ§Ш®ШӘ ШұШ§ Ш¬ШҜШ§ЪҜШ§ЩҶЩҮ Ш§ШөЩ„Ш§Шӯ Ъ©ЩҶЫҢШҜ.

Ш«ШЁШӘ ЩҫШұШҜШ§Ш®ШӘ ЩҲ Ш§ШҙШӘШұШ§Ъ©

ШӘЩҶШёЫҢЩ… Ш§ЩҶЩӮШ¶Ш§ (Ш§ШөЩ„Ш§Шӯ Ш§ШҙШӘШЁШ§ЩҮ)

Ш§ЪҜШұ ЩҫШұШҜШ§Ш®ШӘЫҢ ШұШ§ ШӯШ°ЩҒ Ъ©ШұШҜЫҢШҜ ЩҲ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢШҜ Ш§ЩҶЩӮШ¶Ш§ ШұШ§ Ъ©Щ… Ъ©ЩҶЫҢШҜШҢ Ш§ЫҢЩҶШ¬Ш§ ШӘШ§ШұЫҢШ® Ш¬ШҜЫҢШҜ ШұШ§ ШЁЪҜШ°Ш§ШұЫҢШҜ ЩҲ ШЁЩҮвҖҢШұЩҲШІ Ъ©ЩҶЫҢШҜ.

ШӘШәЫҢЫҢШұ ШұЩ…ШІ Ъ©Ш§ШұШЁШұ (Ш§ШҙШӘШұШ§Ъ©)

ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ ШҜШіШӘШұШіЫҢ (Щ…ЩҲШЁШ§ЫҢЩ„ Ш§ЩҒШұШ§ШҜЫҢ Ъ©ЩҮ В«Ш®ЫҢШұВ» ШІШҜЩҶШҜ)

Ш§ЫҢЩҶ Ш§ЩҒШұШ§ШҜ ШҙЩ…Ш§ШұЩҮ Щ…ЩҲШЁШ§ЫҢЩ„ Ш®ЩҲШҜ ШұШ§ Ш«ШЁШӘ Ъ©ШұШҜЩҮвҖҢШ§ЩҶШҜШӣ Ъ©ШҜ Ъ©Ш§ШұШЁШұЫҢ ШўЩҶвҖҢЩҮШ§ ЩҮЩ…Ш§ЩҶ ШҙЩ…Ш§ШұЩҮ Ш§ШіШӘ. ЩҫШі Ш§ШІ ЩҲШ§ШұЫҢШІШҢ Ъ©Ш§ШұШЁШұ ШұШ§ ШЁШ§ ЩҮЩ…ЫҢЩҶ Щ…ЩҲШЁШ§ЫҢЩ„ Ш§Ш¶Ш§ЩҒЩҮ ЩҲ {{ subscription_price }} ШҜЩ„Ш§Шұ (Ъ©Щ„ ШЁШіШӘЩҮ) Ш«ШЁШӘ Ъ©ЩҶЫҢШҜ.

Щ…ЩҲШЁШ§ЫҢЩ„ (Ъ©ШҜ Ъ©Ш§ШұШЁШұЫҢ) ШӘШ§ШұЫҢШ® ШҜШұШ®ЩҲШ§ШіШӘ

ЩҲШұЩҲШҜ ЪҜШұЩҲЩҮЫҢ ЩҶШ§Щ… ЩҲ ЩҶШ§Щ… Ш®Ш§ЩҶЩҲШ§ШҜЪҜЫҢ (Ъ©Ш§ШұШЁШұШ§ЩҶ ЩӮШЁЩ„ЫҢ)

ШҜШұ ЩҮШұ Ш®Ш·: Щ…ЩҲШЁШ§ЫҢЩ„ШҢ ЩҶШ§Щ…ШҢ ЩҶШ§Щ… Ш®Ш§ЩҶЩҲШ§ШҜЪҜЫҢ (ШЁШ§ Ъ©Ш§Щ…Ш§ ЫҢШ§ ЩҲЫҢШұЪҜЩҲЩ„ Ш¬ШҜШ§ Ъ©ЩҶЫҢШҜ). ЩҒЩӮШ· Ъ©Ш§ШұШЁШұШ§ЩҶЫҢ Ъ©ЩҮ Щ…ЩҲШЁШ§ЫҢЩ„ШҙШ§ЩҶ ШҜШұ Ш¬ШҜЩҲЩ„ ЩҮШіШӘ ШЁЩҮвҖҢШұЩҲШІ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.

Щ„ЫҢШіШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ ЩҲ ЩҲШ¶Ш№ЫҢШӘ Ш§ШҙШӘШұШ§Ъ©

ШҙЩҶШ§ШіЩҮ Щ…ЩҲШЁШ§ЫҢЩ„ ЩҶШ§Щ… ЩҲ ЩҶШ§Щ… Ш®Ш§ЩҶЩҲШ§ШҜЪҜЫҢ Ш§ЩҶЩӮШ¶Ш§ ЫҢШ§ШҜШҜШ§ШҙШӘ Ш№Щ…Щ„ЫҢШ§ШӘ

ШўШ®ШұЫҢЩҶ ЩҫШұШҜШ§Ш®ШӘвҖҢЩҮШ§

Ъ©Ш§ШұШЁШұ (ЩҶШ§Щ… / Щ…ЩҲШЁШ§ЫҢЩ„) Щ…ШЁЩ„Шә ШӘШ§ШұЫҢШ® ЩҫШұШҜШ§Ш®ШӘ Ъ©ШҜ ЩҲШ§ШұЫҢШІ ЩҲЫҢШұШ§ЫҢШҙ ШӯШ°ЩҒ

ЩҲЫҢШұШ§ЫҢШҙ ЩҫШұШҜШ§Ш®ШӘ

ЩҲЫҢШұШ§ЫҢШҙ ЩҶШ§Щ… ЩҲ ЩҶШ§Щ… Ш®Ш§ЩҶЩҲШ§ШҜЪҜЫҢ

ШӯШ¶ЩҲШұ ЩҲ ЩҒШ№Ш§Щ„ЫҢШӘ ШЁШ§ШІШҜЫҢШҜЪ©ЩҶЩҶШҜЪҜШ§ЩҶ

ШЁШұ Ш§ШіШ§Ші Щ„Ш§ЪҜ IP ЩҲ ШІЩ…Ш§ЩҶ ЩҲШұЩҲШҜ. ШұЩҲЫҢ В«Ш¬ШІШҰЫҢШ§ШӘВ» ЩҮШұ IP ШЁШІЩҶЫҢШҜ ШӘШ§ ШұЩҲШІЩҮШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҲ ШіШ§Ш№Ш§ШӘ Ъ©Ш§ШұЪ©ШұШҜ ШӘЩӮШұЫҢШЁЫҢ ШұШ§ ШЁШЁЫҢЩҶЫҢШҜ.

ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҶЩ…ЩҲЩҶЩҮ (ЩҒЩҮШұШіШӘ + Introduction + Rights + Who We Are + Canada's History + Govern + Federal Elections + Canadian Symbols + Justice + Modern Canada + Exam1вҖ“3)

ШЁШ§ШІШҜЫҢШҜ Ш§ЫҢЩҶ ШЁШ®Шҙ ШЁЩҮ ШӘЩҒЪ©ЫҢЪ© ШұЩҲШІ (ЫіЫ° ШұЩҲШІ Ш§Ш®ЫҢШұ)

ШӘШ§ШұЫҢШ®ЩҒЩҮШұШіШӘ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§IntroductionRightsWho We AreCanada's HistoryGovernFederal ElectionsCanadian SymbolsJusticeModern CanadaExam1Exam2Exam3Ш¬Щ…Ш№ ШұЩҲШІ

ШЁШ§ШІШҜЫҢШҜ ШЁЩҮ ШӘЩҒЪ©ЫҢЪ© ШұЩҲШІ (ЫіЫ° ШұЩҲШІ Ш§Ш®ЫҢШұ)

ШӘШ§ШұЫҢШ®ШӘШ№ШҜШ§ШҜ ШЁШ§ШІШҜЫҢШҜIP ЫҢЪ©ШӘШ§

IPЩҮШ§ЫҢ ЩҫШұШӘШ№ШҜШ§ШҜ

IPШҜШіШӘЪҜШ§ЩҮШӘШ№ШҜШ§ШҜ ШЁШ§ШІШҜЫҢШҜШ§ЩҲЩ„ЫҢЩҶ ШЁШ§ШІШҜЫҢШҜШўШ®ШұЫҢЩҶ ШЁШ§ШІШҜЫҢШҜШұЩҲШІЩҮШ§ЫҢ ЩҒШ№Ш§Щ„Ш¬ШІШҰЫҢШ§ШӘ

Ш¬ШІШҰЫҢШ§ШӘ IP:

ШӘШ§ШұЫҢШ®Ш§ЩҲЩ„ЫҢЩҶ ШЁШ§ШІШҜЫҢШҜШўШ®ШұЫҢЩҶ ШЁШ§ШІШҜЫҢШҜШӘШ№ШҜШ§ШҜЩ…ШҜШӘ (ШҜЩӮЫҢЩӮЩҮ)
